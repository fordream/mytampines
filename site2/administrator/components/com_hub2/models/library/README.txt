Windows Azure Samples for PHP

Enables PHP developers to easily add Windows Azure to their web applications

Project URL: 
http://phpazure.codeplex.com/ 

Project Detail:
Using Windows Azure SDK for PHP, these PHP samples show how PHP developers can
take advantage of the Microsoft Cloud Services Platform - Windows Azure. This 
includes samples for Windows Azure Blobs, Tables and Queues.

In addition, PHP solutions for SQL Azure and Windows Azure Cloud Drives (X-Drives)

Technologies:
PHP; Windows Azure
